# Causeway

Static SPA for Groundwire comet identity operations. No backend; browser talks
directly to urb-watcher snapshot, mempool.space, and the Groundwire RPC/faucet.
All signing happens on an air-gapped hardware wallet via a paste/QR round-trip —
no keys ever enter the browser.

Design: `/Users/trent/.claude/plans/sharded-kindling-parasol.md`.

## Dev

```
npm install
npm run dev          # localhost:5173
npm test             # vitest — 47 tests across encoder, cue, patp, signing
npm run build        # emits into ../website/static/causeway/
```

The companion page `/hw-stub.html` is a dev-only software HW wallet. Paste a
`SignRequest` JSON, enter a BIP-39 mnemonic or `@q` seed, and it prints a
`SignResponse`. **Never use with a real mnemonic.**

## Status

Working today:

- `protocol/encoder.ts` — full arm-by-arm port of `lib/urb-encoder.hoon`;
  bit-exact against 20 golden vectors from `tests/dump_vectors.py`.
- `oracle/cue.ts` — iterative Hoon jam decoder; 14 round-trip vectors pass.
- `oracle/state.ts` — decodes `+$state` from `sur/urb.hoon` into typed
  `UrbState` with `unvIds: Map<@p, Point>`.
- `oracle/snapshot.ts` — fetches urb-watcher's `application/x-urb-jam` body
  and hands back `UrbState`.
- `oracle/point.ts` — `lookupPoint` + `resolveAuthPubkey` (via mempool.space).
- `protocol/patp.ts` — `@p` ↔ bigint via `urbit-ob`.
- `chain/bip86.ts` — BIP-341 `tapTweak`.
- `chain/tapscript.ts` — urb-tagged Taproot leaf wrapper.
- `chain/{mempool,rpc,sponsor}.ts` — REST clients.
- `signing/qr-protocol.ts` — `SignRequest`/`SignResponse` types + JSON codec.
- `signing/verify.ts` — Schnorr + nonce + count checks.
- `signing/hw-stub.ts` — functional dev-only software HW wallet page.
- `ops/*` — one module per operation (login, rekey, escape, cancel-escape,
  adopt, reject, detach, fief, set-mang, spawn). Each builds the correct
  `skim-sotx` and wraps it in a urb-tagged leaf script.
- `ui/app.ts` — hash-router UI for login → dashboard → per-op pages.

Not yet implemented (clearly stubbed, not faking success):

- `chain/commit.ts`, `chain/reveal.ts` — tx assembly. Port remaining from
  `gw-onboard.py` lines ~1024-1168. Ops currently produce `SignRequest`s with
  empty `sighashes`; `finalize()` throws.
- Reveal-witness parsing for script-path `authPubkey` (see
  `oracle/point.ts` TODO — currently returns tweaked key for both slots).
- Real camera-based QR scanning + multi-frame BBQR — UI is paste-based.
- Native comet-miner integration — `spawn/miner.ts` is a stub.

## Testing

```

`tests/dump_vectors.py` (encoder) and `tests/dump_jam_vectors.py` (cue).
Regenerate after changes:

python3 tests/dump_vectors.py
python3 tests/dump_jam_vectors.py
npm test
```
